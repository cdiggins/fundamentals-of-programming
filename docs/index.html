The Fundamentals of Programming

This web-site is dedicated to helping people learn and teach the fundamental concepts of programming using JavaScript and graphics.

What is Programming?

Programming is the act of performing computation (computing values) and automation (telling computers and robots to do things) with code.

What are the Fundamental Parts of Code?

The key components of code are:

  • Variables
  • Expressions
  • Functions
  • Data Types

What is an Expression?

An expression is either a literal value (e.g., the number 42), a reference to a variable (e.g. x), an operator ("+") applied to one or more sub-expressions, or a function call (e.g., f(x)).

What is a Variable and a Value?

A variable is a name that is associated with a value. A value is a piece of data that the computer program is tracking at run-time. It might be a number or a string, or an aggregation of multiple values.

Variables can be associated with (assigned) new values. If a variable is assigned a value when it is created, this is called "initialization".
